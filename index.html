/claude-ai Claude.ai essentials

The core features of the Claude app on web, desktop, and mobile — many people use Claude daily without knowing half of these exist.

Projects

A workspace that groups related chats and gives Claude persistent knowledge: upload files, add custom instructions, and every chat inside knows them.

How: Sidebar → Projects → New Project → add files & instructions. On Team plans, projects can be shared.

Artifacts

When Claude builds something — a webpage, a game, a diagram, an app — it appears in a live side panel you can run, edit, remix, and publish with a shareable link. Published artifacts can now even pull live data through each viewer’s own connectors.

How: Ask Claude to "make an interactive…" anything. Click Publish to share it. Browse yours at claude.ai/artifacts.

Skills

Reusable instruction packs that teach Claude how to do a task your way (brand guidelines, report formats, workflows). Claude loads them automatically when relevant.

How: Settings → Capabilities → Skills, or ask Claude to create a skill for you. Invoke directly with /skill-name.

Memory

Claude remembers key details across conversations — your role, preferences, ongoing work — so you don't repeat yourself. You can view and edit everything it remembers.

How: Settings → Memory. Ask "what do you remember about me?" anytime. Use incognito chats for conversations that should leave no trace.

Research & Deep Research

Claude searches the web (and your connected apps), reads dozens of sources, and writes a cited report. Deep research runs longer for harder questions.

How: Click the Research button in the chat box. Good prompts: "Compare X and Y in depth."

Connectors (MCP)

Plug Claude into your tools — Gmail, Google Calendar, Drive, Slack, Notion, Microsoft 365 (now with write access) and hundreds more — so it can read and act on your real data.

How: Settings → Connectors → browse the directory. Then just ask: "Summarize this week's unread email."

Extended thinking

Lets Claude reason step-by-step before answering — dramatically better for math, code, and tricky logic. Sonnet 5 adapts its thinking depth automatically.

How: Toggle thinking in the chat box's tools menu (where available), or ask Claude to "think carefully step by step."

Voice mode

Talk with Claude hands-free on mobile. Great for brainstorming on a walk or dictating instead of typing.

How: Mobile app → tap the voice icon next to the chat box.

Styles

Change how Claude writes — Concise, Explanatory, Formal — or create a custom style from a writing sample so replies always sound like you.

How: Click the style picker in the chat box → Create & edit styles.

File uploads & data analysis

Drop in PDFs, images, screenshots, or spreadsheets. Claude reads them, and its analysis tool can run real code to crunch CSVs and chart the results.

How: Drag files into the chat box (or paste a screenshot). For spreadsheets, ask for specific charts or trends.

Claude Design 2026

A dedicated mode for visual work — generate polished designs, mockups, and prototypes you can iterate on conversationally.

How: Look for Design in the tools menu, or ask Claude to design a screen/poster/prototype.

Chat search & history

Every past conversation is searchable — and Claude itself can search your chat history to recall what you discussed weeks ago.

How: Use the search bar in the sidebar, or ask Claude: "Find our conversation about the marketing plan."

/cowork Cowork & automation

Claude doesn't just chat anymore — it works alongside you and keeps going when you step away.

Cowork Now on web & mobile

Give Claude a folder and a goal; it works through real files — organizing, drafting, building spreadsheets — checking in as it goes. Sessions now follow you across desktop, web, and mobile.

How: Open Cowork in the Claude app, point it at a folder, describe the outcome you want. Approve steps from your phone.

Scheduled tasks

Ask Claude to do something on a schedule — a Monday-morning news digest, a weekly report draft — and it runs automatically.

How: In Cowork or Claude Code, ask: "Every weekday at 9am, summarize my inbox." Manage from the tasks/routines view.

Claude in Slack

Tag Claude inside a Slack thread to delegate a task — it reads the context and gets to work, no app-switching.

How: Install the Claude Slack app, then @Claude in any conversation.

Background work

Long tasks keep running while you close the tab. Claude notifies you — including phone push notifications — when it finishes or needs a decision.

How: Start a long task and walk away. Enable notifications in the mobile app to get pinged.

/claude-code Claude Code

Anthropic's agentic coding tool — in your terminal, desktop app, IDE, and browser. It reads your codebase, edits files, runs commands, and ships features. Here's the full command deck.

Getting started

Install once, then run claude in any project folder. Available in terminal, Desktop app, VS Code/JetBrains, and on the web at claude.ai/code.

How: curl -fsSL https://claude.ai/install.sh | bash (or npm install -g @anthropic-ai/claude-code), then run claude in a project. /init generates a CLAUDE.md project guide.

CLAUDE.md — project memory

A markdown file Claude reads at the start of every session: build commands, conventions, warnings. The single highest-leverage customization.

How: /init creates one. Press # during a session to quickly save a note to memory. Edit anytime with /memory.

Auto mode 2026

The middle ground between approving every action and skipping all permissions: a safety classifier lets safe actions run uninterrupted and blocks risky ones.

How: Cycle permission modes with Shift+Tab, or manage rules in /permissions. Available on Pro and up.

Effort levels & fast mode

Dial how hard the model thinks: higher effort for gnarly problems, fast mode for quick output at lower cost.

How: /effort xhigh for the hardest tasks; /fast on for speed; /model to switch models.

Subagents & background sessions

Claude can spawn specialized agents that run in the background (they can even spawn their own). Your session keeps moving while they work.

How: /agents to create custom agents; claude agents shows every session — running, blocked, done.

Hooks, MCP & plugins

Deep customization: hooks run your scripts on tool events, MCP servers connect external tools, plugins bundle it all for your team.

How: /hooks, /mcp (or claude mcp login from your shell), /plugin to browse & install plugins.

Cloud power: Ultraplan & code review

Draft plans in the cloud and review them in a web editor; or launch a fleet of bug-hunting review agents on your branch.

How: /code-review ultra for the multi-agent cloud review; Routines on claude.ai/code fire scheduled cloud agents.

Keyboard power moves Pro tip

Esc interrupts Claude mid-task. Double-Esc rewinds to edit an earlier message. Shift+Tab cycles permission modes (including plan mode).

How: Customize everything in /keybindings. Vim fan? /vim.

/settings Settings worth changing

Five minutes in settings makes Claude noticeably better for you.

Pick your model

Sonnet 5 (default on Pro) balances speed and power with a 1M-token context. Opus 4.8 and the Claude 5 family are there for the hardest work on higher plans.

How: Model picker at the top of any chat.

Turn on (and curate) memory

Memory makes every conversation smarter — and it's fully inspectable. Review what Claude knows and delete anything you'd rather it forget.

How: Settings → Memory → view/edit. Per-project memory keeps work contexts separate.

Quiet hours & break reminders New

Part of Claude Reflect: set hours when Claude won't notify you, and get gentle reminders during long sessions.

How: Settings → Reflect / Notifications.

Connect your tools

The single biggest upgrade for daily usefulness: connect Gmail, Calendar, and Drive, and Claude becomes a real assistant instead of a chatbot.

How: Settings → Connectors. Start with calendar + email.

Feature previews

Anthropic ships experiments behind a toggle. Turning previews on gets you tomorrow's features today.

How: Settings → Feature previews (availability varies by plan).

Data & privacy controls

Control whether chats can improve models, export your data, and use incognito chats for anything sensitive.

How: Settings → Privacy. Incognito: the ghost icon on a new chat.

/hidden-gems Hidden gems

The tricks people wish someone had told them about sooner.

Gem Paste screenshots anywhere

Screenshot an error, a design, a whiteboard — paste it straight into any Claude chat (even the terminal). Claude reads images remarkably well.

Gem Ask Claude about Claude

The most underused feature: just ask. "What can you do with my calendar?" "How do I make you more concise?" Claude knows its own features.

Gem Edit & branch old messages

Edit any earlier message in a chat to branch the conversation from that point — perfect for trying a different approach without starting over.

Gem Dictation beats typing

Long, rambling voice-dictated prompts often work better than short typed ones — more context means better answers. Don't polish your prompts.

Gem Turn chats into artifacts

Any explanation can become a shareable interactive page: "Turn this into an interactive study guide / quiz / visualization" then hit Publish.

Gem Shell mode in Claude Code

Type ! npm test and Claude runs it and explains the output — no second prompt needed.

Gem Interrupt without losing work

In Claude Code, Esc stops Claude mid-action so you can redirect — it keeps everything done so far. Way better than waiting out a wrong turn.

Gem Project instructions > repeating yourself

If you correct Claude the same way twice, put it in Project instructions (claude.ai) or CLAUDE.md (Claude Code). It's permanent from then on.

/community-tips What power users swear by

Tips collected from community write-ups, newsletters, and forum threads — the tricks people actually share with each other. Third-party tools are marked; use your judgment.

Pro tip Feed Claude markdown, not PDFs

PDFs, Word docs, and slide decks carry hidden formatting that burns context tokens. The same content as plain markdown is often half the size — and Claude reads it better.

How: Export to markdown, or convert with Microsoft's free markitdown tool (third-party workflow).

Pro tip Review with fresh eyes

Don't ask the session that wrote the code to review it — it's biased toward its own choices. A clean session (or /code-review, which spawns fresh reviewer agents) catches more.

How: /clear, then ask for a review — or just run /code-review.

Pro tip Watch Claude work

A live view of every file Claude opens, every tool call, and its reasoning makes long tasks much easier to trust — and to interrupt at the right moment.

How: Press Ctrl+O in Claude Code to toggle the verbose stream.

Pro tip Layer your CLAUDE.md files

Keep personal preferences in ~/.claude/CLAUDE.md, project conventions in the repo's CLAUDE.md, and folder-specific rules in nested files — each loads only where it applies.

How: /memory shows exactly which memory files are loaded right now.

Pro tip Script Claude with headless mode

Claude Code runs without a UI for automation: pipe in a prompt, get structured output back. People wire it into CI, cron jobs, and shell scripts.

How: claude -p "your task" --output-format json, with --allowedTools to scope what it may do.

Pro tip Put stable context first

Prompt caching rewards structure: keep the unchanging parts (architecture notes, API docs) at the top of your prompt or project instructions, and the question at the end. Faster and cheaper.

Pro tip The MCP starter kit

Three community-favorite MCP servers: Playwright (drive a clean browser for testing), Chrome DevTools (performance traces, network logs), and Firecrawl (turn web pages into clean markdown).

How: /mcp to add servers (all third-party — review what you install).

Pro tip Talk instead of typing

Voice dictation matches thinking speed — power users brain-dump long, messy instructions by voice and let Claude sort them out. Works in Claude Code too.

How: /voice in Claude Code; the mic icon on mobile and desktop apps.

Pro tip Run Claude from anywhere

Start a cloud coding session from your phone in a waiting room, then /teleport it into your laptop later — or leave a session running at home and steer it remotely with /remote-control.

Pro tip Read any repo like a book

For understanding an unfamiliar GitHub project, community users swap github.com for deepwiki.com in the URL to get auto-generated docs and architecture diagrams to feed Claude (third-party service).

Pro tip Know what things cost

Agentic features are hungrier than chat — community estimates put Cowork-style file tasks at several times normal token use. Check before you're surprised.

How: /usage breaks down consumption by skill, subagent, plugin, and MCP server.

Pro tip Make Claude teach you

The fastest way to learn all of this isn't a cheat sheet (even this one) — it's /powerup's interactive lessons, and asking Claude itself: "what should I be using that I'm not?"
